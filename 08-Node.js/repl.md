# repl

> 交互式解释器（REPL）既可以作为一个独立的程序运行

## 命令与特殊键

- .break
> ctrl c 组合件，终止表达式的继续输入
- .clear
- .exit
> 关闭输入输入出流
- .help
> 显示所有的帮助列表
- .save
- .load
- .editor

- 认的解释器会把最近一次解释的表达式的结果赋值给变量 _ （下划线）
