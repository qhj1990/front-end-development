# react-hook

想要复用一个有状态的组件太麻烦。 react16.8新增hooks

## hooks解决的问题

类组件的不足

* 组件之间复用状态逻辑很难
* 趋向负责难以维护
* this指向

hook上的优势

* 无需改变组件结构的情况下复用状态逻辑
* 能将组件中相互关联的部分拆分成更小的函数

## useEffect

effect: 指那些没有发生在数据向视图转换过程中的逻辑，如 ajax 请求、访问原生dom 元素、本地持久化缓存、绑定/解绑事件、添加订阅、设置定时器、记录日志等
