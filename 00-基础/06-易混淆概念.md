# 易混淆概念

常常有人把库和框架搞混在一起。

## library库

有两种，方法库和UI库
- 方法库library例如：jquery，lodash
- UI库：element ui ant-design

## Framework 框架

Framework用来构建一个应用的主体。
- Angular
- Ember
- Backbone

## toolkit 工具集

- Bootstrap

## Boilerplate 模板

- Ant Design Pro

## Scaffolding 脚手架

Scaffolding的作用是辅助快速搭建项目，但Scaffolding不是最终成果的一部分。
- create-react-app
- vue-cli
- ng-cli
