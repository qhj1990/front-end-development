# BEM

在小型宣传网站上，css如何写不是很重要，但是大型的，复杂的项目，长期的项目如何写css显得尤为重要。
这至少会影响到三个方面:

1. 要花费多长时间去编写代码
2. 需要多少代码
3. 浏览器要加载多少代码量

BEM 是一种css代码风格。Block Element Modifier

## BEM优点

* 块的样式不依赖于同页面的其他元素
* 可复用
* 结构化，结构很清晰

## 资源

* [BEM官网](http://getbem.com/)

## 解析

### Block块

有意义的，独立的实体

### Element元素

块的一部分，没用独立的含义

### Modifier修饰符

Block或者Element的标记，用来改变外观和行为：例如颜色，disable
