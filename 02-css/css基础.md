# css基础

## css盒模型Box model

* content 内容
* padding 内边距
* border 边框
* margin 外边距

box-sizing属性

* 标准盒子模型(content-box),现代浏览器 width, height = content;
* IE盒子模型(border-box), width, height = content + padding + border;

几乎所有的css框架都使用border-box
